Haskell is extremely expressive functionally, which allows for wonderful gems like the three-line Fibonacci sequence

```haskell
fibSlow :: Int -> Int
fibSlow 0 = 0
fibSlow 1 = 1
fibSlow n = fibSlow (n - 1) + fibSlow (n - 2)
```
